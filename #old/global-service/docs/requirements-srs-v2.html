KR PostgreSQL 인스턴스에 CREATE SCHEMA gs. Prisma previewFeatures = ["multiSchema"] + 모든 모델 @@schema("gs"). gs schema 외 권한 회수. pg_dump --schema=gs로 글로벌 데이터만 백업 가능.
